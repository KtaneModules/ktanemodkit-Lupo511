On the Subject of More Complicated Wires

TO-DO: Text

  • Each wire has a LED above it that can either be on or off.
  • Each wire has a space below it that can read "A", "B" or be blank.
  • For each color/LED combination, use the Venn diagram below to find the correct section, then find the first rule that applies in that section to decide whether or not to cut the wire.
  • The Venn diagram is based on the additive color model to figure out color components.
  • If the bomb as a Stereo RCA port, invert the results (i.e. only cut the wires you don't have to cut).

Section A
If the text says "A" If the bomb has more than 3 batteries, cut the wire.
If the alarm in the room is currently active, don't cut the wire.
If the bomb has more than 5 modules (not including needy modules), don't cut the wire.
If none of the above apply, cut the wire.
If the text says "B" If the last digit of the serial number matches the number of blue wires, don't cut the wire.
If the number of already cut wires is odd, cut the wire.
If there is no red wire, cut the wire.
If none of the above apply, don't cut the wire.
If the text is empty If there are less than 2 black wires, cut the wire.
If the bomb has one or more button modules, and if any wire has the color of any button, don't cut the wire.
If every wire is of the same color, cut the wire.
If none of the above apply, cut the wire.
